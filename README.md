1) PRACAI Society Website – Web Development Project

 Project Overview

This project is a fully responsive and interactive website created as part of the Web Development Society selection task.

The website represents the PRACAI Society and includes multiple sections such as:

* Home / Landing Page
* About Us
* Features
* Courses Offered
* Frequently Asked Questions (FAQ)
* Still Got Questions (Contact Prompt)

The goal of this project is to demonstrate basic front-end development skills using HTML, CSS, and JavaScript.



## Technologies Used

* HTML5 → Structure of the website
* CSS3 → Styling, layout, colors, responsiveness
* JavaScript → Interactivity and dynamic behavior



→ Project documentation

# Interactivity (JavaScript)

The website includes:

* Button click alerts
* Enrollment confirmation
* FAQ toggle animation
* Smooth scroll navigation

All interactivity is handled via `script.js`.



# How To Run The Website

1. Download / Clone the project folder
2. Open the folder in **VS Code** or any editor
3. Double-click `index.html`




# Images Usage

All images are stored inside the 'images' folder and linked in HTML using:
